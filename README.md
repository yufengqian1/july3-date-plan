
# july3-date-plan

A tiny static GitHub Pages date-plan page.

## Files

- `index.html` — main page
- `assets/` — self-contained SVG image placeholders
- `.nojekyll` — keeps GitHub Pages from trying to run Jekyll

## Deploy

1. Upload all files in this folder to the root of the repo.
2. Go to Settings → Pages.
3. Choose: Deploy from a branch → main → /root.
4. Open the Pages URL once GitHub finishes deploying.
